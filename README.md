**专为 Alpine Linux 优化，支持 OpenRC 服务管理，使用 apk 安装依赖**  
**支持 Hysteria2、AnyTLS、Shadowsocks-2022（2022-blake3-aes-256-gcm）三种协议**  
**支持交互式自定义监听端口，特别适合 NAT 类型 VPS（端口受限）**  
**无需任何提前准备，只需要一台公网 IPv4 的 Alpine Linux VPS 即可一键部署**

📌 **一键安装命令** (复制以下命令使用`root`直接运行):
```bash
bash <(curl -fsSL https://github.com/Murgoa/mihomo-server-alpine/raw/refs/heads/main/install.sh)
```
📌 **一键卸载命令** (复制以下命令使用`root`直接运行):

```bash
bash <(curl -fsSL https://github.com/Murgoa/mihomo-server-alpine/raw/refs/heads/main/uninstall.sh)
```
